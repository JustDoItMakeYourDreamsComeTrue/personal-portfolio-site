# Персональный сайт-визитка

Современный персональный сайт-портфолио, созданный с использованием React, TypeScript и Vite.

## Технологии

-   React 18
-   TypeScript
-   Vite
-   Tailwind CSS
-   Framer Motion
-   Radix UI

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build
```
